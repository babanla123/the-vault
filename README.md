A simple digital asset management system with a form for users to register an asset(images,documents, etc), store asset data(CID,ownwer address) on solana, using IPFS for file uploads and show registered assets in a dashboard, The front-end was built using React.js for a responsive and user-friendly interface, while the back-end leverages Solana blockchain for secure, decentralized data storage and transaction handling. Smart contracts were written using Rust and deployed on Solana to manage asset records and user permissions securely, IPFS or the InterPlanetary File System which is a decentralized, peer-to-peer network for storing and sharing files, IPFS uses a system where content is addressed by a unique hash called a Content Identifier (CID),This makes data more resilient.
