A simple digital asset management system with a form for users to register an asset(images,documents, etc), store asset data(CID,ownwer address) on solana, using IPFS for file uploads and show registered assets in a dashboard, the software was built using react for the frontend, rust for basic smart contract, anchor framework, IPFS or the InterPlanetary File System which is a decentralized, peer-to-peer network for storing and sharing files, IPFS uses a system where content is addressed by a unique hash called a Content Identifier (CID),This makes data more resilient, tested on solana devnet.
